## بناء جملة Syntax
لإخراج نص في لغة R، يُستخدم إما علامات الاقتباس المفردة أو المزدوجة:
```
"Hello World!"
```
لإخراج الأرقام، يكفي كتابة الرقم مباشرة دون استخدام علامات الاقتباس:
```
5
10
25
```
ولإجراء عمليات حسابية بسيطة، يمكن جمع الأرقام معًا على النحو الآتي:
```
5 + 5
```
